Project Overview

This project is an interactive Excel dashboard developed to analyze the Indian motorcycle market using a dataset of 1,000+ bike records. The dashboard enables users to explore bike performance, pricing, mileage, and segment-wise trends through dynamic charts and slicers.

Key Insights
Premium segment bikes have the highest average ex-showroom prices.
Honda and TVS offer strong mileage performance across models.
Significant pricing differences exist between factory and ex-showroom prices.
High-performance bikes are concentrated in the premium segment.
